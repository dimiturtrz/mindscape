- 2026-07-03 — **SHIN 2018 N-BACK DATASET DEEP-DIVE: RAW FACTS FOR METHOD SELECTION** — Shin 2018 (n=26, 0/2/3-back, 234 trials/class); BenchNIRS cross-subject 5-fold baseline: LDA **38.9%**, CNN **39.3%**, LSTM **34.4%** (all vs 33.3% chance, only LDA/CNN significant) [S14]. Sample-size reality: fNIRS DL overfits n<30 without regularization; LSTM underperforms LDA on Shin (34.4% < 38.9%); random-split CV inflates by 25–40 pts vs true LOSO; BenchNIRS consensus = LDA wins cross-subject, DL requires data augmentation to compete on small N [S14, S20-S22]. Recent SOTA (2023-2025): fNIRS-T 78.22% (different task, not Shin 2018), fNIRSNet 6.58% gain (within-subject only), CNN-LSTM 88.69% (pilot workload, not n-back) [S19, S23-S24]. **VERDICT:** No published cross-subject fNIRS DL method beats Shrinkage-LDA (38–40%) on Shin 2018 n-back yet. Prioritize: (1) Shrinkage-LDA baseline, (2) domain adaptation if <45%, (3) fNIRS-T with rigorous LOSO validation if baseline sufficient.
